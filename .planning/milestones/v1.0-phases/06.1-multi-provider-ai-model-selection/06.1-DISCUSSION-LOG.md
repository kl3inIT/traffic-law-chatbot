## ChatClient Factory

| Option | Description | Selected |
|--------|-------------|----------|
| Dynamic per-request | Đọc active param set mỗi request | |
| Map<String,ChatClient> startup | Pre-build tất cả model lúc khởi động | ✓ (với biến thể) |
| ChatClient duy nhất + options override | Override model per-call | |

**User's choice:** Map<String, ChatClient> build lúc startup từ YAML model catalog. Tuy nhiên, user quyết định **không dùng enum** và **không đọc DB per-request cho chat model** — model đến từ request body (modelId optional), evaluatorModel đọc từ active AiParameterSet.

**Notes:** User giải thích model thay đổi ít, muốn giữ catalog trong YAML (linh hoạt hơn enum, không recompile khi thêm model). `AllowedModel` enum sẽ bị xóa.
