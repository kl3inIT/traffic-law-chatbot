  ```java
  @Service @RequiredArgsConstructor @Slf4j
  public class ChatLogService {
      private final ChatLogRepository chatLogRepository;
      public void save(String question, ChatAnswerResponse response, GroundingStatus groundingStatus,
                       String conversationId, int promptTokens, int completionTokens, int responseTime) {
          ChatLog log = ChatLog.builder()
              .question(question)
              .answer(response.getAnswer())
              .sources(buildSourcesString(response))
              .groundingStatus(groundingStatus)
              .conversationId(conversationId)
              .promptTokens(promptTokens)
              .completionTokens(completionTokens)
              .responseTime(responseTime)
              .build();
          chatLogRepository.save(log);
      }
      private String buildSourcesString(ChatAnswerResponse response) {
          if (response.getSources() == null) return null;
          return response.getSources().stream()
              .map(s -> s.getTitle() != null ? s.getTitle() : s.getUrl())
              .collect(java.util.stream.Collectors.joining(", "));
      }
      public Page<ChatLog> findAll(Pageable pageable) { return chatLogRepository.findAll(pageable); }
      public Optional<ChatLog> findById(UUID id) { return chatLogRepository.findById(id); }
  }
  ```

Create ChatLogAdminController:
  ```java
  @RestController
  @RequestMapping
  @RequiredArgsConstructor
  public class ChatLogAdminController {
      private final ChatLogService chatLogService;

      @GetMapping(ApiPaths.CHAT_LOGS)
      public Page<ChatLogResponse> list(
              @RequestParam(required=false) GroundingStatus groundingStatus,
              @RequestParam(required=false) @DateTimeFormat(iso=ISO.DATE_TIME) OffsetDateTime from,
              @RequestParam(required=false) @DateTimeFormat(iso=ISO.DATE_TIME) OffsetDateTime to,
              @RequestParam(required=false) String q,
              @PageableDefault(sort="createdDate", direction=Sort.Direction.DESC) Pageable pageable) {
          return chatLogService.findFiltered(groundingStatus, from, to, q, pageable)
              .map(ChatLogResponse::fromEntity);
      }

      @GetMapping(ApiPaths.CHAT_LOG_BY_ID)
      public ResponseEntity<ChatLogDetailResponse> getById(@PathVariable UUID logId) {
          return chatLogService.findById(logId)
              .map(log -> ResponseEntity.ok(ChatLogDetailResponse.fromEntity(log)))
              .orElse(ResponseEntity.notFound().build());
      }
  }
  ```

  Add findFiltered() to ChatLogService that uses Spring Data Specifications or individual repository query methods to combine the optional filters (groundingStatus, dateRange, questionLike). If all params are null, use findAll(pageable). Use JpaSpecificationExecutor<ChatLog> on ChatLogRepository for combined filtering.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Admin client → ChatLog REST | Unauthenticated admin endpoints expose chat content (questions + full answers) — same surface as all other admin APIs in this project |
| ChatService → ChatLogService | Log persistence is a write path that must never block or fail the primary chat response |
| Check runner → ChatService | The check runner calls ChatService.answer() repeatedly — a malformed CheckDef could cause repeated LLM calls |
