## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] base-ui Select onValueChange typed as `(value: string | null, ...) => void`**
- **Found during:** Task 1 (TypeScript compile check)
- **Issue:** `setSelectedModelId` accepts `string` but `onValueChange` passes `string | null`. TypeScript error TS2322.
- **Fix:** Wrapped callback: `(val) => setSelectedModelId(val ?? '')` — converts null to empty string (treated as "no model selected")
- **Files modified:** `app/(chat)/page.tsx`, `app/(chat)/threads/[id]/page.tsx`
- **Commit:** 0cf7a17 (included in same commit)
